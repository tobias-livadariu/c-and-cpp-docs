# C++ Study Guide

This repo is a compact source of truth for rebuilding practical C++ knowledge, with an emphasis on LeetCode, data structures and algorithms, and eventually competitive programming.

The current focus is C++, not C. The docs use C++17 as the default baseline because it is widely supported by online judges, with C++20+ notes where they are useful.

## How To Use This Repo

1. Start with [cpp/00-learning-roadmap.md](cpp/00-learning-roadmap.md).
2. Read the crash course chapters in order.
3. Work through [cpp/stl](cpp/stl) until you can choose containers and algorithms without guessing.
4. Use [cpp/dsa](cpp/dsa) while solving problems.
5. Use [cpp/practice/quiz-prompts.md](cpp/practice/quiz-prompts.md) with future agents for active recall.

## Layout

- [cpp](cpp): C++ language crash course and learning plan.
- [cpp/stl](cpp/stl): Standard library and STL-focused reference notes.
- [cpp/dsa](cpp/dsa): DSA problem patterns with C++ templates.
- [cpp/practice](cpp/practice): Quiz prompts and teaching scaffolds.
- [cpp/references.md](cpp/references.md): External references used to shape this guide.

## Study Rule

For each topic, do three passes:

1. Read the notes.
2. Rebuild the example without looking.
3. Solve one small problem where the topic is necessary.
